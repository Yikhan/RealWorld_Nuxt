# Realworld - Nuxt.js

已部署到阿里云上

[访问地址](http://47.74.68.207:3000/)

基于开源项目Realworld - Conduit论坛

<https://github.com/gothinkster/realworld>

实现全部基本功能

- 用户登录/注册
- 发布文章
- 发布评论
- 点赞/取消点赞文章
- 关注/取消关注用户
- 查看所有的/关注用户的/特定标签的文章
- 用户详情页
- 过滤无效的空标签
- 处理无效的用户头像

运行命令配置于 `package.json`

```json
"dev": "nuxt"
"build": "nuxt build"
"start": "nuxt start"
```

运行`npm run dev`或者`yarn dev`即可直接启动项目
